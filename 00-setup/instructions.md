# Instructions

## Check Java Version

❯ java --version
java 10.0.2 2018-07-17
Java(TM) SE Runtime Environment 18.3 (build 10.0.2+13)
Java HotSpot(TM) 64-Bit Server VM 18.3 (build 10.0.2+13, mixed mode)

❯ which java
/usr/bin/java

❯ javac --version
javac 10.0.2

❯ which javac
/usr/bin/javac

## Compile and Run Java Code

```
$ javac App.java && java App
```
